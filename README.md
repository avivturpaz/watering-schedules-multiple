# Multi-Plant Watering Tracker

> **Retired Venture experiment — historical reference only.**

A small Flask application for tracking several plants, their watering frequency, last-watering date, next due date, and watering history.

## What is implemented

- Plant and species records
- Configurable watering intervals
- Due, due-soon, and overdue status calculation
- Watering event log
- SQLite persistence
- Paddle checkout and Plausible integration points

## Status

Development stopped on July 10, 2026. The application is not maintained or production-ready. Watering schedules are user-entered reminders and are not horticultural recommendations.

Created as part of the archived Venture product-experiment portfolio.
